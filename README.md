# Inventara Parvaldibas Sistema

Inventāra pārvaldības sistēma ir Java Swing lietotne, kas palīdz pārvaldīt dažādas preces noliktavā.

## Funkcionalitāte
- [x] Preču pievienošana (nosaukums, kategorija, stāvoklis un daudzums)
- [x] Preču dzēšana
- [x] Inventāra saraksta skatīšana tabulā
- [x] Datu kārtošana (augošā un disltošā secībā)
- [x] Datu glabāšana teksta failā
- [ ] Datubāzes atbalsts
- [ ] Preču rediģēšana
